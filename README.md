# কিনকিপার 👫

কিনকিপার হলো অর্থপূর্ণ সম্পর্কগুলোর জন্য একটি ব্যক্তিগত "শেলফ"। এটি ইন্টারঅ্যাকশন ট্র্যাক করে এবং যোগাযোগের লক্ষ্য নির্ধারণ করার মাধ্যমে আপনাকে সবচেয়ে গুরুত্বপূর্ণ সম্পর্কগুলো ব্রাউজ করতে, যত্ন নিতে এবং লালন করতে সাহায্য করে।

## 🚀 ফিচারসমূহ

- **ফ্রেন্ড ড্যাশবোর্ড**: আপনার সমস্ত বন্ধুদের একটি ভিজ্যুয়াল গ্রিড, যেখানে স্ট্যাটাস ইন্ডিকেটর (অন ট্র্যাক, অলমোস্ট ডিউ, ওভারডিউ) এবং ট্যাগ থাকে।

- **বিস্তারিত প্রোফাইল**: দুই-কলামের লেআউট, যেখানে বন্ধুর তথ্য, সম্পর্কের লক্ষ্য এবং পরিসংখ্যানের বিস্তারিত বিবরণ রয়েছে।

- **কুইক চেক-ইন**: আপনার সম্পর্কগুলোকে উষ্ণ রাখতে এক ক্লিকেই কল, টেক্সট বা ভিডিও চ্যাট লগ করুন।

- **ইন্টারঅ্যাকশন টাইমলাইন**: আইকন এবং তারিখসহ আপনার অতীতের ইন্টারঅ্যাকশনের একটি সম্পূর্ণ ইতিহাস।

- **ফ্রেন্ডশিপ অ্যানালিটিক্স**: আপনার যোগাযোগের অভ্যাসের ভিজ্যুয়াল বিশ্লেষণ।

- **রেসপন্সিভ ডিজাইন**: মোবাইল, ট্যাবলেট এবং ডেস্কটপ ভিউয়ের জন্য সম্পূর্ণরূপে অপ্টিমাইজ করা।

## 🛠️ টেকনিক্যাল স্ট্যাক

- **ফ্রন্টএন্ড**: Next.js (অ্যাপ রাউটার), React, Tailwind CSS।

- **আইকন**: Lucide React ও FontAwesome।

- **ডেটা**: বন্ধুদের প্রোফাইলের জন্য JSON-ভিত্তিক স্টোরেজ এবং টাইমলাইন সংরক্ষণের জন্য LocalStorage।
